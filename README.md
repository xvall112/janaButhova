# Jana Buthova - A Web Application for Fitness Exercises
This project is a web application built for fitness enthusiasts, coached by Jana Buthova. The application provides a database of video exercises and type of workouts that users can attend it to improve their physical fitness. 

## Features
Jana Buthova offers the following features:

## A homepage with various workout to choose from.
- Detailed information about each workout.
- Information about Jana
- Price List
- Contact form

## Technology
Jana Buthova is built using the following technologies:

- Gatsbyjs
- Contntful
-Material-UI
